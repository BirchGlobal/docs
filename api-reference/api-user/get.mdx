---
title: 'Get API User'
api: 'GET /api/v1/api-users/me'
---

This endpoint retrieves the details of the current API user.

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/api-users/me`

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/api-users/me" \
     --header "Authorization: Bearer <token>"

```

</RequestExample>


Response
The response returns a JSON object containing the details of the API user.



<ResponseExample>
````md ResponseExample Syntax
{

  "data": {
    "id": "api-user-id",
