This endpoint lists all runs for a specific agent.

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/agents/:agent_id/runs`

### Path Parameters

- `agent_id`: The ID of the agent whose runs are to be listed.

<RequestExample>

```bash
curl --url "https://api.gptlab.cloud/api/v1/agents/:agent_id/runs" \
     --header "Authorization: Bearer <token>" \
     --header "Accept: application/json"
```

</RequestExample>


Response
The response returns a JSON object containing a list of datasources associated with the agent.
