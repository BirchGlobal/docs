---
title: 'Add Datasource to Agent'
api: 'POST /api/v1/agents/:agent_id/datasources'
---

This endpoint adds a datasource to a specific agent.

## HTTP Request

`POST https://api.gptlab.cloud/api/v1/agents/:agent_id/datasources`

### Headers

- Content-Type: application/json
- Authorization: Bearer `<token>`
- Accept: application/json

### Path Parameters

- `agent_id`: string (The ID of the agent to which the datasource is being added)

### Request Body

The request expects the following JSON object:

```json
{
  "datasourceId": "string" // The ID of the datasource to be added
}

## Response

The response returns a JSON object indicating the success of the operation and details of the agent to which the datasource was added.
