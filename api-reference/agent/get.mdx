This endpoint retrieves the details of a single agent.

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/agents/:agent_id`

### Path Parameters

- `agent_id`: The ID of the agent to retrieve.

<RequestExample>

```bash
curl --url "https://api.gptlab.cloud/api/v1/agents/:agent_id" \
     --header "Authorization: Bearer <token>" \
     --header "Accept: application/json"
```

</RequestExample>


Response
The response returns a JSON object containing the details of the agent.
