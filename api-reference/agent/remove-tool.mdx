title: 'Remove Tool from Agent'
api: 'DELETE /api/v1/agents/:agent_id/tools/:tool_id'
---

This endpoint removes a specific tool from an agent.

## HTTP Request

`DELETE https://api.gptlab.cloud/api/v1/agents/:agent_id/tools/:tool_id`

### Path Parameters

- `agent_id`: The ID of the agent from which the tool will be removed.
- `tool_id`: The ID of the tool to be removed from the agent.

<RequestExample>

```bash
curl -X DELETE \
     --url "https://api.gptlab.cloud/api/v1/agents/:agent_id/tools/:tool_id" \
     --header "Authorization: Bearer <token>" \
     --header "Accept: application/json"
```

</RequestExample>

Response
The response to this request will typically indicate the success or failure of the removal operation. The actual response may vary depending on the API's implementation.
