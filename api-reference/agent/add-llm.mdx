---
title: 'Add LLM to Agent'
api: 'POST /api/v1/agents/:agent_id/llms'
---

This endpoint adds a Language Model (LLM) to an agent.

## HTTP Request

`POST https://api.gptlab.cloud/api/v1/agents/:agent_id/llms`

### Path Parameters

- `agent_id`: The ID of the agent to which the LLM will be added.

### Headers

- Content-Type: application/json
- Authorization: Bearer `<token>`

<RequestExample>

```bash
curl -X POST \
     --url "https://api.gptlab.cloud/api/v1/agents/:agent_id/llms" \
     --header "Content-Type: application/json" \
     --header "Authorization: Bearer <token>" \
     --data '
{
  "llmId": "your-llm-id"
}
```
</RequestExample>

Request Body
The request body must include:

- llmId: String (required) - The ID of the LLM to be added.


Response
The response will return a JSON object indicating the success of the operation and details of the LLM added to the agent.
