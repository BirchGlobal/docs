---
title: 'Create Agent'
api: 'POST /api/v1/agents'
---

This endpoint creates a new agent.


<RequestExample>

```bash
curl -X POST \
     --url "https://api.gptlab.cloud/api/v1/agents" \
     --header "Content-Type: application/json" \
     --header "Authorization: Bearer <token>" \
     --header "Accept: application/json" \
     --data '
{
  "name": "string",
  "llmModel": "string",
  "description": "string"
}
```
</RequestExample>



## HTTP Request

`POST https://api.gptlab.cloud/api/v1/agents`

### Headers

- Content-Type: application/json
- Authorization: Bearer `<token>`
- Accept: application/json

### Request Body

The request body should contain an object with the following fields:
- `isActive`: Optional boolean
- `name`: String (required)
- `initialMessage`: Optional string
- `prompt`: Optional string
- `llmModel`: String (required)
- `description`: String (required)
- `avatar`: Optional string
