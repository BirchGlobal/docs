This endpoint retrieves a list of all tools.

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/tools`

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/tools" \
     --header "Authorization: Bearer <token>"
```

</RequestExample>

### Response
The response returns a JSON object containing a list of all tools with their details.
