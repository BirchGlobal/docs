---
title: 'List Datasources'
api: 'GET /api/v1/datasources'
---

This endpoint retrieves a list of all datasources.

### HTTP Request

`GET https://api.gptlab.cloud/api/v1/datasources`

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/datasources" \
     --header "Authorization: Bearer <token>"
```

</RequestExample>

### Response
The response returns a JSON object containing a list of all datasources with their details.
