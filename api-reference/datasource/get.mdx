This endpoint retrieves the details of a specific datasource.

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/datasources/:datasource_id`

### Path Parameters

- `datasource_id`: The ID of the datasource to retrieve.

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/datasources/:datasource_id" \
     --header "Authorization: Bearer <token>"
```

</RequestExample>
