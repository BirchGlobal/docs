This endpoint retrieves a list of all Large Language Models (LLMs).

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/llms`

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/llms" \
     --header "Authorization: Bearer <token>"
```

</RequestExample>

### Response
The response returns a JSON object containing a list of all LLMs with their details.




<ResponseExample>
````md ResponseExample Syntax
{
  "success": true,
  "data": [
    {

      ],
      "createdAt": "ISO 8601 datetime",

      "apiUserId": "api-user-id",
