This endpoint retrieves details of a specific Large Language Model (LLM).

## HTTP Request

`GET https://api.gptlab.cloud/api/v1/llms/:llm_id`

### Path Parameters

- `llm_id`: The ID of the LLM to retrieve.

<RequestExample>

```bash
curl -X GET \
     --url "https://api.gptlab.cloud/api/v1/llms/:llm_id" \
     --header "Authorization: Bearer <token>"
```

</RequestExample>



### Response
The response returns a JSON object containing the details of the specified LLM.
